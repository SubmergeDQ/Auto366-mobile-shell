# Auto366 答案提取工具

从天学网移动应用的加密数据中解密并提取答案。

## 功能特性

- ✅ Root 权限检查
- ✅ 交互式目录选择（两级导航）
- ✅ 纯Shell实现（无需Python）
- ✅ 智能答案提取（完整内容而非选项字母）
- ✅ 自动去重 + 题号排序
- ✅ HTML标签清理

## 使用前提

1. **Android设备** (已root)
2. **已安装天学网应用** 并至少打开过一次需要查看的课程
3. **MT管理器** 或支持root执行脚本的终端

## 快速使用

### MT管理器（推荐）

```
1. 将 Auto366.sh 传输到手机存储
2. 找到 Auto366.sh → 点击该文件
3. 勾选「以root身份执行」和「使用系统环境执行」→ 点击执行
4. 按提示选择目录，等待完成
5. 查看生成的 answer.txt
```

### 终端/Termux

```bash
su
cd /sdcard/你的文件夹/
chmod +x Auto366.sh
./Auto366.sh
# 调试模式: ./Auto366.sh --debug
```

## 命令行参数

| 参数 | 说明 |
|------|------|
| `(无)` | 正常运行 |
| `-d, --debug` | 调试模式 |
| `-h, --help` | 帮助信息 |

## 常见问题

**Q: 提示需要root权限**
A: MT管理器中勾选「以root身份执行」，或终端先输入 `su`

**Q: 未找到OpenSSL等工具**
A: 确保 MT管理器 选择「使用系统环境执行」

**Q: Flipbook目录不存在**
A: 确保已安装up366并浏览过课程

**Q: 答案数量不对**
A: 多文件会自动去重，相同题号+内容的只保留一个
