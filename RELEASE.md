# Auto366 发行说明

## v1.0.0 (2026-04-28) - 首个发行版

### 功能特性

- ✅ Root 权限检查
- ✅ 交互式目录选择（两级导航）
- ✅ 纯 Shell 实现（无需 Python）
- ✅ 智能答案提取（完整内容而非选项字母）
- ✅ 自动去重 + 题号排序
- ✅ HTML 标签清理

### 核心功能

1. **环境检测**
   - 自动检测 OpenSSL、Python、xxd 等工具
   - 支持多种解密方式（OpenSSL/Python）
   - 提供详细的错误提示和解决方案

2. **目录导航**
   - 交互式选择课程/试卷目录
   - 两级目录选择界面
   - 自动扫描数字命名的文件夹

3. **文件处理**
   - 自动复制加密文件到临时目录
   - 批量解密 .u3enc 文件
   - 从解密后的 JS 文件中提取答案

4. **答案提取**
   - 提取完整答案内容（不只是选项字母）
   - 自动去重（相同题号 + 内容只保留一个）
   - 按题号数字顺序排序
   - 清理 HTML 标签

### 使用方法

#### MT 管理器（推荐）

```
1. 将 Auto366.sh 传输到手机存储
2. 找到 Auto366.sh → 点击该文件
3. 勾选「以 root 身份执行」和「使用系统环境执行」→ 点击执行
4. 按提示选择目录，等待完成
5. 查看生成的 answer.txt
```

#### 终端/Termux

```bash
su
cd /sdcard/你的文件夹/
chmod +x Auto366.sh
./Auto366.sh
# 调试模式：./Auto366.sh --debug
```

### 命令行参数

| 参数 | 说明 |
|------|------|
| `(无)` | 正常运行 |
| `-d, --debug` | 调试模式（显示详细日志和文件预览） |
| `-h, --help` | 显示帮助信息 |

### 输出格式

脚本运行后在同目录下生成 `answer.txt`：

```
========================================
  答案
========================================
第 1 题：「She is cold.」
第 2 题：「By bus.」
...
第 20 题：「In the 19th century.」

========================================
  日志
========================================
[OK] 合并去重后：20 个答案
```

### 系统要求

- Android 设备（已 root）
- 已安装天学网应用并至少打开过一次课程
- MT 管理器或支持 root 执行脚本的终端
- OpenSSL 或 Python（带 pycryptodome 库）
